图解排序算法：选择排序

发表于 2017-04-02 | 分类于 数据结构与算法

选择排序

选择排序是一种简单直观的排序算法。它的工作原理是每一次从待排序的数据元素中选出最小（或最大）的一个元素，存放在序列的起始位置，直到全部待排序的数据元素排完。

算法原理

原理见含义：步骤以对给定长度为n的数组a[]从小到大排序为例。

  1. 从待排序元素第1位a[i]与待排序元素第2位a[i+1]开始比较，记录较小元素下标为index。(注意是待排序元素第一位第二位)
  2. 上一步骤的较小值a[index]与下一位a[2]比较，记录较小的元素下标为index。
  3. ……
  4. 上一较小值a[index]与末位a[n-1]相比，记录较小的元素下标为index。
  5. 交换a[index]与本轮起始位a[i]交换数值。

以上为一循环的步骤，每循环一次可以确定该次待排序元素间中的最小值，并将其置于合适位置。

图解说明

我们注意到，长度为n=5的数组a[]，共排序了4次。我们以int类型 i 表示排序次序的话，则 i 的取值范围在 1<= i <= n - 1。因数组元素操作其实是操作其下标，而下标从0开始，因此为方便后续的比对及交换，我们将 i 的取值范围减1，即 0 <= i <= n - 2 （此式与前式中，i 的取值范围是一样的）。又因n-1其实是数组末位的下标，因此按照约定俗成的写法，我们将式子变为 0 <= i < n - 1 （此式与前2个，i 的取值范围依然是相同的）。因此，有了这样的代码：

接下来，我们观察每次大循环内部是如何找出该次的最小值的。

第一次(i=0)大循环中，从a[0]与a[1]相比，标记较小值元素下标为index，一直和数组最末位a[n-1]比较，然后最小值a[index]与a[0]交换数值。

第二次(i=1)大循环中，从a[1]与a[2]相比，标记较小值元素下标为index，一直和数组最末位a[n-1]比较，然后最小值a[index]与a[1]交换数值。

第三次(i=2)大循环中，从a[2]与a[3]相比，标记较小值元素下标为index，一直和数组最末位a[n-1]比较，然后最小值a[index]与a[2]交换数值。

第四次(i=3)大循环中，从a[3]与a[4]相比，标记较小值元素下标为index，一直和数组最末位a[n-1]比较，然后最小值a[index]与a[3]交换数值。
